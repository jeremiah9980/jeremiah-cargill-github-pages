# Teleo Title static capture (reconstructed)

This folder contains a **manual static reconstruction** of parts of Teleo Title's current public website for offline review and iteration.

Why reconstructed instead of a raw mirror:
- I was able to access page text through web extraction and view screenshots shared in chat.
- I was **not** able to directly pull the raw site HTML/assets from the execution environment.
- The pages here are intended as a tweakable working copy for design/content suggestions, not a byte-for-byte mirror.

Included:
- `index.html` — reconstructed homepage
- `company.html` — reconstructed company/team page
- `styles.css` — shared styling

Suggested next step:
- Compare these pages against the live site in-browser and adjust copy, spacing, CTAs, and structure.
